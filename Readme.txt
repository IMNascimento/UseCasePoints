Fazer o sistema de calculo esforço tempo de Gustav Karner
Feito em O.O.

1- Para instalar ele vai na pasta config/
2- Arquivo chamado estrutura.php.
3- Coloque os seus dados nas variaveis do construct.
4- Depois execute esse arquivo php "ele irá subir o banco de dados para você".
5- Depois basta ir para pagina inicial e navegar e realizar os calculos.
